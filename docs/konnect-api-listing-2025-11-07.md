# Kong Konnect API Listing

**Source**: https://developer.konghq.com/api/
**Date**: 2025-11-07

## Konnect APIs

There are approximately **24 distinct Konnect APIs**, organized by category:

### Core Platform APIs
- **Identity Management (v3)** - Users, teams, organizations, and permissions
- **Organization Management (v3)** - Organizational-level settings
- **Audit Logging (v2)** - Security monitoring and compliance
- **Notification Hub (v1)** - Notification management

### Developer Portal & API Management
- **Portal Management (v3)** - AIP-compliant portal automation
- **Developer Portal (v3)** - Portal user data access
- **API Builder (v3)** - Managing APIs for Developer Portal (beta)
- **Application Auth Strategies (v2)** - Authentication for APIs
- **Service Catalog (v1)** - Service catalog

### Runtime Management
- **Control Planes (v2)** - Control plane management
- **Control Planes Config (v2)** - Services, routes, certificates
- **Dedicated Cloud Gateways (v2)** - Infrastructure configuration
- **Event Gateway (v1)** - Kafka protocol proxy
- **Mesh Manager (v0)** - Global mesh control plane

### Analytics & Monitoring
- **Analytics Dashboards (v2)** - Dashboard tiles with saved queries
- **API Request (v2)** - Near real-time request records
- **Metrics Endpoint (4.0)** - Metrics extraction

### Misc
- **Consumers (v1)** - Centralized consumer management
- **Search (v1)** - Konnect search functionality
- **CMEK (v1)** - Customer Managed Encryption Keys
- **Kong Identity (v1)** - Beta identity features

## Notes

- API Products (v2) is deprecated and not included in this listing
- Each API provides OpenAPI specifications and Insomnia integration for testing
- Version numbers range from v0 (beta) through v3, with some using semantic versioning
